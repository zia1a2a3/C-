### 1.标识符的命名规则
- 标识符不能是关键字
- 标识符是由字母，数字，下划线构成
- 标识符的第一个字母只能是字母或下滑线
- 标识符是区分大小写的

   ***标识符要尽量做到见名知意***

### 2.赋值运算符

### 4.指针的定义与应用
